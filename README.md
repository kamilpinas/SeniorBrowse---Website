# SeniorBrowse — Website

The marketing landing page for **SeniorBrowse**, a browser extension that makes
the internet simple for the people you love. A caregiver sets it up once — a calm
home screen, big buttons, and a helper panel on every page — and the senior just
browses.

The site is a single long-scroll page with hand-drawn illustration, a live
interactive product demo, and scroll-driven animation throughout.

---

## What's on the page

The page reads top to bottom as one continuous story:

- **Scroll progress bar** — a thin indicator across the top that tracks how far
  down the page you are.

- **Top bar** — the SeniorBrowse brand, quick links (How it works, Safety, Free,
  FAQ), and an **Add to Browser** button pointing at the Chrome Web Store listing.

- **Hero — "You set it up. They just browse."** — the opening pitch beside a
  hand-drawn illustration of a grandmother at her laptop. Her eyes follow the
  cursor, she blinks, and her head tilts slightly — all animated live.

- **See it in action — "Watch the side panel do the work."** — a self-playing,
  fully interactive demo. A ghost cursor drives a real helper side panel (Home,
  Back/Forward, Volume, Move Page, Text Size, Save Page, Fullscreen, Refresh,
  Close) across mock pages — news, home screen, video, photos, and email. You can
  take over and click around yourself at any time. Below it, a 60-second
  walkthrough video with a "Follow along" checklist of the setup steps.

- **Before & after — "The same page — without the noise."** — a drag slider that
  wipes between a cluttered website and the same page cleaned up by SeniorBrowse.

- **A note from the founder — "Dear friend,"** — a short personal letter about the
  founder's grandmother and why the internet wasn't built for people like her.

- **Safety & Control** — four built-in protections, each with a small looping
  demonstration: no ads or clutter, scam links blocked before they're clicked, no
  accidental downloads, and a browsing history kept privately on the senior's own
  computer.

- **Getting started — "Three steps. Done in five minutes."** — the caregiver setup
  flow shown as interactive step cards (add it and name them, customise the theme,
  text size and safety, then walk through the tutorial together).

- **Free — "It's free."** — the pricing section: no trial, no card, no account,
  with an optional pay-what-you-want note for anyone who wants to chip in.

- **FAQ — "Questions caregivers actually ask."** — an expandable list covering
  learning curve, bookmarks, activity log, privacy, device support, remote setup,
  cost, and uninstalling.

- **Final call to action — "Hold the door open for them."** — the closing pitch,
  a last Add to Browser button, and the site footer.

## Built with

- **React 18** + **TypeScript**
- **Vite 6** for the dev server and build
- **GSAP** (with `@gsap/react` and ScrollTrigger) for the animations and the
  interactive demo
- **Lenis** for smooth scrolling and section snapping
- **Phosphor Icons** for iconography
